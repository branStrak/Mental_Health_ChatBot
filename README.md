##Mental Health Chatbot

A simple AI-powered chatbot for mental health support. It offers empathetic chats, mood tracking, and links to resources. Focuses on privacy and ethics. Not a substitute for professional help.

##Quick Start

Clone and run:
bashgit clone https://github.com/yourusername/mental-health-chatbot.git
cd mental-health-chatbot
pip install -r requirements.txt
python app.py
Visit http://localhost:5000.

##Features

Empathetic conversations using AI.
Crisis detection with hotline links.
Two different models to fulfill needs.
Local data storage for privacy.
All features are planned or basic.

##Tech Stack

Python with Flask.
Hugging Face for NLP.
PostgreSql for data.
 

##Installation

Clone repo.
Create virtual env: python -m venv venv and activate.
pip install -r requirements.txt.
Set .env if needed (e.g., API keys).
Run python app.py.

##Usage

Chat via web: Say "I'm depressed" for responses. View dashboard for insights.
